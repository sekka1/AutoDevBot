Sign Up Process

  1. 1. Sign Up

    Thanks! Your message has been successfully sent!
    Ooops! There was an error sending your message.

  2. 2. There is no step #2

    You are done signing up. The following steps below are all automated steps performed by AutoDevBot to start monitoring your API.

  3. Automatic Github Repo Creation

    We wil automatically create a Github repository for you to use with AutoDevBot. This will hold your monitoring code. We will automatically populate this repository with some working sample code that will start monitoring once you finish signing up.

  4. Docker.io Container Creation

    We will build a Docker.io container that has the Github repository we created for you and will only contain your monitoring code in it.

  5. Deploy Container

    Your Docker.io container is deployed out to all of our locations and started up. Monitoring will begin immediately. The same process happens everytime there is a commit into this repository.
